[KanjiCode] menu

When TTXKanjiMenu plugin is installed, [KanjiCode] menu is added to Tera Term.
Menu display character code for Japanese or Korean environment selected in settings. Sending/receiving character code and Ambiguous Character Width can be changed using menus.

Ambiguous Character Width is disabled when the character code is not UTF-8.

Setting dialog for this plugin is opend selecting [Setting] from menu.
