ウィンドウサイズ変更プラグイン

Tera Term のメニューに [Resize] メニューを追加する。

機能

TTXResizeMenu プラグインを入れると、Tera Term のメニューに [Resize] メニューを追加する事が出来ます。
[Resize] メニュー内の項目が選択されると、そのサイズに Tera Term の VT ウィンドウのサイズが変更されます。

使用方法

プラグインを入れただけでは [Resize] メニューは表示されません。
設定ファイルにメニュー項目を登録する必要があります。

メニュー内の項目を変更したい場合は、TERATERM.INI に [Resize Menu] セクションを作成し、"ResizeMenuN = X, Y" の形式で指定して下さい。N は 1 からの連番です。
X または Y のいずれかが 0 の時は、その要素は変更されません。例えば 0, 37 の時は、横幅は変更せずに縦幅を 37 行に変更します。
X または Y の値を -1 にした場合、画面サイズを超えない範囲での最大値として扱われます。
メニューの項目数は最大で 20 です。

設定例

[Resize Menu]
;
; 横幅 80桁, 縦幅 37行
ResizeMenu1 =  80, 37
;
; VTの標準サイズ
ResizeMenu2 =  80, 24
;
; 横幅は変更せずに、縦幅を52行にする。
ResizeMenu3 =   0, 52
;
; 縦幅は変更せずに、横幅を132桁にする。(VTの132桁モード(DECCOLM)相当)
ResizeMenu4 = 132,  0
