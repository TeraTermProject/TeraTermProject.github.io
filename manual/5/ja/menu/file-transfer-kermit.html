Kermit ([File] Transfer メニュー)

Kermit プロトコルでファイルを転送します。

Receive
ファイルを受信します。 その前にホスト側の Kermit の Send コマンドを実行してください。
Get...
ホスト側の Kermit が server の状態の時に、Tera Term からホスト側のファイル名を指定してファイルを受信します。
Send...
ファイルを送信します。その前にホスト側の Kermit の Receive コマンドまたはServer コマンドを実行してください。
Finish
ホスト側が server の状態のとき、それを解除します。
