[Control] メニュー

Reset terminal
ターミナルの各種モードをリセットします。
Reset remote title
リモートタイトルをリセットします。
Are you there
Telnet の AYT (Are You There) コマンドを送出します。 通常ホストはこれに呼応して何等かのメッセージを送り返してきます。 Telnet 接続のときだけ使用できます。
Send break
ブレーク信号を送出します。sendbreak マクロコマンドも参照。
Reset port
シリアルポートをリセットします。 シリアルポート接続のときだけ使用できます。
Broadcast command
ブロードキャストコマンドを送信するためのダイアログを表示します。
Open TEK
TEK window を開きます。
Close TEK
TEK window を閉じます。
Macro
マクロを実行します。
Show Macro Window
実行中のマクロウィンドウを表示します。
