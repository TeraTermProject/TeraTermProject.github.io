SSH Forwarding ([Setup] メニュー)

"TTSSH: Forwarding" ダイアログボックス

SSH 通信を使ってポート転送を行う設定ができます。
SSH 経由で暗号化して、既存のプロトコルをセキュアな通信を行わせる場合に指定します。

Port Forwarding

Add...
Forward する方向、ポート番号を指定して一覧に追加します。
Edit...
選択された Forwarding の設定を編集します。
Remove...
選択された Forwarding の設定を削除します。

X Forwarding

Display remote X application on local X server
このチェックボックスをオンにすると、サーバ上の X クライアントとPC上の X サーバの間の通信が暗号化されます。

"TTSSH: Forwarding Edit" ダイアログボックス

各転送設定を編集する。

local forwarding

Forward local port
転送するローカル側のポート番号
listen
ローカル側のリッスンアドレス。省略すると localhost が使われる。
to remote machine
転送先のホスト名
port
転送先のポート番号

remote forwarding

Forward remote server port
転送するリモート側のポート番号
listen
リモート側のリッスンアドレス。省略すると localhost が使われる。
SSH1プロトコルでは利用されない。
to local machine
転送先のホスト名
port
転送先のポート番号

dynamic forwarding

Forward dynamic port
ダイナミック転送するローカル側のポート番号。SOCKS Proxy として動作する。
listen
ローカル側のリッスンアドレス。省略すると localhost が使われる。
