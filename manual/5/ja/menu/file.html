[File] メニュー

New connection...
新しいホストまたはシリアルポート、名前付きパイプに接続します。
Duplicate session
現在のセッションを複製します。 TELNET, SSH, Cygwin 接続のみが複製されます。
Log...
受信される文字をファイルに書き込みます。
Pause Logging
ログの記録を一時中断します。
Comment to Log...
ログファイルへコメントを追加することができます。 追加できるコメントは最大255文字までです。
View Log
ログファイルを開きます。
Show Log dialog...
ログダイアログを表示します。
Stop Logging
ログの記録を終了します。
Send file...
ファイルを送信します。
選択したとき
"Send file" ダイアログ が表示されます。
CTRL を押しながら選択したとき
"Send file" ダイアログがスキップされて、ファイル選択ダイアログが表示されます。
前回と同じ送信オプションが使用されます。
設定を変更することで 動作を入れ替えることができます。
Transfer
Kermit/XMODEM/YMODEM/ZMODEM/B-Plus/Quick-VAN プロトコルでファイルを転送します。
SSH SCP
SCP プロトコルでファイルを転送します。
Change directory...
ファイル転送のためのディレクトリを変更します。
Replay Log...
バイナリログを再生します。/R オプションと等価です。
Print...
現在の画面または、マウスで選択された文字を印刷します。
Disconnect
接続を切ります。 TCP/IP 接続時で Auto window close オプションが選択されている場合、Tera Term を終了します。
Exit
Tera Term を終了します。 ホストまたはシリアルポートに接続されている場合は、接続を切ります。
Exit All
すべての Tera Term を問い合わせせずに終了します。 ホストまたはシリアルポートに接続されている場合は、接続を切ります。
