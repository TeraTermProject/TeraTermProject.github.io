TTProxy コマンドライン

形式

TTProxy では、以下のコマンドラインパラメータが使用できます。 

-proxy=protocol://<user<:password>@>proxyserver:proxyport
protocol://<user<:password>@>proxyserver:proxyport/realhost
-noproxy
protocol
プロキシのプロトコル
以下の種類が使用可能です。
  • http
  • telnet
  • socks4
  • socks5
  • none
proxyserver
プロキシサーバ
proxyport
プロキシサーバのポート番号
realhost
実際に接続したいサーバ
user
プロキシで認証が必要な場合のユーザ名
password
プロキシで認証が必要な場合のパスワード
-noproxy
プロキシ設定を無効化する。-proxy=none:// と同じ。

使用例

ttermpro.exe -proxy=socks5://user:pass@proxy:1080 sshserver /ssh
ttermpro.exe -proxy=telnet://proxy:8023 telnetserver:23 /nossh
ttermpro.exe -proxy=none:// sshserver /ssh
ttermpro.exe -noproxy sshserver /ssh

注意

以下の形式は使用できません。 

protocol://<user<:password>@>proxyserver:proxyport/realhost:ポート番号

Tera Term が処理する telnet://server<:port> 形式のパラメータと競合するため、プロキシサーバを指定するために以下の形式を使用することはできなくなりました。 

protocol://<user<:password>@>proxyserver:proxyport
