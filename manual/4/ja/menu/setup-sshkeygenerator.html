SSH Keygenerator ([Setup] メニュー)

"TTSSH: Key Generator" ダイアログボックス

詳しい解説はSSH 接続の「キーの生成」 をご覧下さい。

Key type
生成する鍵の種類を指定します。
RSA1
SSH1用 RSA鍵を生成します。
RSA
SSH2用 RSA鍵を生成します。
DSA
SSH2用 DSA鍵を生成します。
鍵長が1024ビット以外のDSA鍵は、SSHプロトコルとして非正規の鍵であり、サーバによっては使えない可能性がありますので作成できません。
DSA鍵の鍵長を1024ビットより大きくしても、セキュリティを高める事にはなりません。 より強度の高い鍵を使いたい場合は、RSA鍵やECDSA鍵を使用して下さい。
ECDSA-256
SSH2用 ECDSA(256bit)鍵を生成します。
ECDSA-384
SSH2用 ECDSA(384bit)鍵を生成します。
ECDSA-521
SSH2用 ECDSA(521bit)鍵を生成します。
ED25519
SSH2用 ED25519鍵を生成します。
Key Bits
生成する鍵の鍵長を指定します。
bcrypt KDF
秘密鍵フォーマットとして bcrypt KDF を指定します。
Generate
鍵を生成します。
Key passphrase
生成した秘密鍵に付けるパスフレーズを指定します。
Confirm passphrase
秘密鍵に付けるパスフレーズの確認をします。 Key passphraseと同じものを入力してください。
Save public key
生成した公開鍵を保存します。
Save private key
生成した秘密鍵を保存します。
秘密鍵は AES-128 で暗号化されます。(Tera Term 4.76(TTSSH 2.63) までは 3DES で暗号化されました)
