動作環境

Tera Term は以下の OS で動作します。

  • Microsoft Windows 95 (*1) (*2) (*3) (*4)
  • Microsoft Windows 98, 98 Second Edition
  • Microsoft Windows Me
  • Microsoft Windows NT 4.0 SP6 (*2) (*4)
  • Microsoft Windows 2000
  • Microsoft Windows XP
  • Microsoft Windows Vista
  • Microsoft Windows Server 2003, 2003 R2
  • Microsoft Windows Server 2008, 2008 R2
  • Microsoft Windows 7
  • Microsoft Windows 8
  • Microsoft Windows Server 2012
  • Microsoft Windows 8.1
  • Microsoft Windows Server 2012 R2
  • Microsoft Windows 10

注1: Windows 3.1 はサポートされていません。
注2: x64 および IA-64 Windows もサポートされています。(WOW64 による)
注3: Tera TermのインストーラはWindows2000未満をサポートしません。(バージョン4.95以降)
注4: CygTerm+はWindows Vista以降のサポートとなります(バージョン4.105以降)。古いWindowsで動作させたい場合は、古いWindowsに対応したCygwinを使ってCygTerm+をソースコードからビルドしてください。
(*1) Windows Sockets2.0 が必要です。
(*2) Internet Explorer 5 以上が必要です。
(*3) LogMeTT, TTLEditor は動きません。
(*4) Tahoma フォントのインストールを推奨します。
