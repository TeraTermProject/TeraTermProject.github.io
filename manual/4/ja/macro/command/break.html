break

ループから抜ける。(バージョン4.53以降) 

break

解説

"for" や "while" のループから抜ける。

例

for i 1 10
  yesnobox 'ループから抜けますか?' 'Tera Term'
  if result break
next

while 1
  recvln
  strcompare inputstr "OK"
  if result=0 then
    break
  else
    sendln "abc..."
  endif
endwhile

参照

  • do, loop
  • for, next
  • until, enduntil
  • while, endwhile
  • continue
