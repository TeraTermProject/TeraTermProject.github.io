foldersearch

フォルダがあるか確かめる。(バージョン4.69以降) 

foldersearch <foldername>

解説

フォルダ <foldername> が存在するかどうか調べる。
存在する場合は、システム変数 result に1が返る。存在しない場合は、result に0が返る。

注意

foldersearch コマンドは、<foldername> がファイルとして存在する場合には0を返す。このため、フォルダ作成前の存在チェックには使用できない。

例

foldersearch 'data'
if result=0 messagebox 'Folder not found.' 'error'

name = 'data'
filesearch name
if result=1 then
    foldersearch name
    if result=1 then
        sprintf2 msg '%s is a folder.' name
        messagebox msg ''
    else
        sprintf2 msg '%s is a file.' name
        messagebox msg ''
    endif
else
    sprintf2 msg '%s is not found.' name
    messagebox msg ''
endif
